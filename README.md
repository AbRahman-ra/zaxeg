# (ZAXEG) ZATCA XML E-Invoice Generator

- Building an invoice generator that comply with ZATCA e-invoicing phase 2 requirements
- After [Trying to understand ZATCA API]() and [Having a look over someone's work]() for e-invoicing 2 years back, and with a bigger knowledge in Java and XML. I decided to get back to the project again, and I am very ambitious I can get something working. However, **Your Contributions are highly appreciated**
